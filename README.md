# Happy Season Back Office
React.js

## Installation

``` bash
# clone the repo
$ git clone https://github.com/Ahmed-Imam/hs-backoffice.git

# go into app's directory
$ cd hs-backoffice

# install app's dependencies
$ npm i
```

## Basic usage

``` bash
# dev server  with hot reload at http://localhost:PORT

$ npm start
```
